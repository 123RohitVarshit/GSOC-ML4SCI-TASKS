# GSOC-ML4SCI-TASKS (QMLHEP)

This repository contains my solutions and implementations for the **Google Summer of Code (GSoC) ML4SCI** evaluation tasks, specifically targeting the **QMLHEP (Quantum Machine Learning in High Energy Physics)** organization.

## Completed Tasks

This repository is organized into distinct directories for each task:

### [Task I: Quantum Circuits](./TASK%20I)
Implementation of foundational Parameterized Quantum Circuits (PQCs). This task explores how classical data can be encoded into quantum states and transformed via rotational and entanglement gates, forming the basis of QML.
- **Code:** [`TASK I/QUANTUM_CIRCUITS.ipynb`](./TASK%20I/QUANTUM_CIRCUITS.ipynb)

### [Task II: Quark-Gluon Jet Tagging with GNNs](./TASK%20II)
A deep learning approach to classifying high-energy particle jets as Quarks or Gluons using Graph Neural Networks (GNNs). This project implements `ParticleNet` (Dynamic EdgeConv) and `JetGAT` (GATv2) to filter out noise and discover spatial correlations within jets.
- **Code Directory:** [`TASK II/`](./TASK%20II/)

### [Task XI: Quantum Multi-Layer Perceptron (QMLP)](./TASK%20XI)
Design and training of a Parameterized Quantum Circuit (PQC) acting as a Quantum Multi-Layer Perceptron. This task demonstrates hybrid quantum-classical training loops optimized for HEP classification challenges.
- **Code:** [`TASK XI/UPDATED_PQC_MLP.ipynb`](./TASK%20XI/UPDATED_PQC_MLP.ipynb)

## About the Organization
**ML4SCI (Machine Learning for Science)** is an open-source organization bridging the gap between advanced ML techniques and fundamental science, particularly High-Energy Physics, Astrophysics, and Quantum physics. The QMLHEP project focuses on utilizing the emerging capabilities of Quantum Computers for data analysis at the Large Hadron Collider (LHC).
