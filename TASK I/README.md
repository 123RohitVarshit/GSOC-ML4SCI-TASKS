# Task I: Quantum Circuits for ML4SCI QMLHEP

This directory contains the implementation for **Task I** of the ML4SCI QMLHEP challenges. The goal of this task is to design and implement fundamental quantum circuits that serve as the building blocks for more advanced Quantum Machine Learning (QML) algorithms.

## Overview
In high-energy physics, simulating interactions and classifying particle events demand immense computational power. Quantum Computing introduces new paradigms (like superposition and entanglement) which give rise to Parameterized Quantum Circuits (PQCs). 

This task includes:
- Building basic quantum states.
- Implementing standard single-qubit and multi-qubit gates.
- Designing parameterized quantum circuits.
- Measuring expectation values.

## Files
- `QUANTUM_CIRCUITS.ipynb`: The main Google Colab / Jupyter Notebook containing the step-by-step code and mathematical explanations of the quantum circuits implemented using frameworks like Pennylane or Qiskit.

## How to Run
Open `QUANTUM_CIRCUITS.ipynb` in Jupyter Notebook or Google Colab and run the cells sequentially. Ensure you have the necessary quantum libraries (e.g., `pennylane`, `qiskit`, `numpy`, `matplotlib`) installed.
